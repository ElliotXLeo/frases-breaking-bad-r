Desarrollado con React, HTML, CSS y JavaScript. Frases de Breaking Bad. Aplicación que cita las frases más famosas de la serie Breaking Bad. La data la consume de una API. Al cargar la página se puede visualizar una frase debido a que se emplea useEffect. Además se puede actualizar la frase mediante un botón. 

## Instalación
- Ubicarse en la carpeta que contendrá el proyecto
- Abrir terminal de comandos
  - git clone https://github.com/ElliotXLeo/frases-breaking-bad-api-jp-r
  - cd frases-breaking-bad-api-jp-r
  - npm install
  - npm start
    - Levanta un servidor con la aplicación
  - npm run build
    - Genera un paquete para el despliegue en producción
